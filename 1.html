HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers.

You can use comments to explain your code, which can help you when you edit the source code at a later date. This is especially useful if you have a lot of code.
